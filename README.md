# 课设要求      
 设计内容               
1.	主要的数据表    
      
1）  英语分级单词表            
2）  常用单词例句表       
3）  短文分级表           
2．	功能模块    
        
1）	实现英语单词的录入、修改、删除等基本操作。   
2）	实现常用英语单词例句的录入、修改、删除等基本操作。    
3）	实现英语单词检索、翻译等。    
4）	常用英语单词例句检索。       
5）	根据难度随机生成一份单词测试题目。       
6）	能够检索出短文极其翻译对照表。     
7）	具有数据备份和数据恢复功能。
